Сгенерируйте PNG иконки через Android Studio:

1. File → New → Image Asset
2. Icon Type: Launcher Icons (Adaptive and Legacy)
3. Foreground Layer: выберите ic_launcher_foreground.xml
4. Background Layer: выберите цвет #0F1218
5. Нажмите Next → Finish

Иконки автоматически создадутся для всех разрешений:
- mdpi: 48x48px
- hdpi: 72x72px
- xhdpi: 96x96px
- xxhdpi: 144x144px
- xxxhdpi: 192x192px
